#Student Manager Frontend
##Описание
Frontend часть системы управления студентами и оценками. Веб-интерфейс для работы с API бэкенда.

##Функциональность
*Управление студентами: просмотр, добавление, редактирование, удаление

*Управление курсами: список курсов, запись студентов

*Выставление оценок: интерфейс для преподавателей

*Фильтрация и поиск: поиск по имени, фильтрация по статусу, GPA

*Отчёты и статистика: визуализация успеваемости, графики

*Панель администратора: управление системой

##Технологии
*React 18 - библиотека для построения интерфейсов

*TypeScript - типизированный JavaScript

*Vite - сборщик проекта (быстрый аналог Create React App)

*Material-UI (MUI) - UI компоненты

*React Router - навигация

*Axios - HTTP клиент для работы с API

*React Query - управление состоянием и кэширование

*Chart.js - визуализация данных (графики)

*Formik + Yup - формы и валидация
